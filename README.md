## ⚡ Smart EV Charging Station (Python)

A Python-based simulation of a **Smart Electric Vehicle (EV) Charging Station** that demonstrates real-world charging logic using priority-based scheduling, grid-aware power management, and safety monitoring.

### 🔍 Overview
The system intelligently manages multiple EVs by evaluating battery level, charging type (AC/DC), grid load, and temperature constraints. It simulates realistic charging behavior, handles power faults and emergency stops, and calculates dynamic pricing based on current load conditions.

### 🚀 Key Features
- Priority-based EV charging queue  
- AC and DC fast-charging simulation  
- Real-time battery temperature and power monitoring  
- Dynamic grid-load–based pricing  
- Emergency stop and fault handling  
- Charging history stored using JSON  

### 🎯 Purpose
This project is designed for **academic use** to demonstrate smart infrastructure concepts, real-time simulation, and object-oriented programming in Python.
